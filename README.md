# MoCCA-Games105: 角色动画与物理仿真

> 北京大学 PKU 角色动画与物理仿真课程 (Games105 / MoCCA) 实验与项目代码

## 📌 课程简介

本仓库包含北京大学《角色动画与物理仿真》课程 (Games105) 的实验作业、课程项目及相关学习资料。课程涵盖角色运动学、关键帧动画、数据驱动动画、物理仿真、强化学习驱动的角色控制等核心内容。

## 🧪 实验内容

| 实验 | 主题 | 说明 |
|------|------|------|
| Lab 1 | 角色运动学基础 | 正向/逆向运动学实现 |
| Lab 2 | 关键帧动画与插值 | 动画曲线与插值方法 |
| Lab 3 | 数据驱动动画 | 运动匹配或相关技术 |
| Lab 4 | 物理仿真 | 刚体/柔体动力学模拟 |
